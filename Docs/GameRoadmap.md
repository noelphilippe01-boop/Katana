# Game Roadmap — Katana

## Terminé

- **Phase 1–3** — Mouvement, combat, ciblage, spawn, loot de base
- **Phase 4** — UI Canvas (menus, pause, HUD, paramètres Audio / Gameplay / Interface)
- **Inventaire (base)** — fenêtre équipement / sac / or, pause à l'ouverture (`I` ou `B`)

## À venir

- **Phase 5** — Assets 3D, types d'ennemis
- **Phase 6** — StatSystem, loot avancé, items dans le sac, équipement complet
- **Phase 7** — Multijoueur coop (voir [NetworkingRoadmap.md](NetworkingRoadmap.md))

## Paramètres (UI)

- [ ] **Menu raccourcis clavier** — onglet ou section dédiée dans les paramètres pour consulter et rebinder les contrôles (déplacement, combat, armes, pause, etc.). La barre de raccourcis permanente du HUD a été retirée en attendant cette feature.
